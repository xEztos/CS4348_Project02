How to compile:
	javac Project2.java

How to run:
	java Project2 <options>
	where possible options include:
		-d, D <int>		number of doctors (default 3)
		-p, P <int> 		number of patients (default 3)
		-s, S <int>		the number of milliseconds to sleep after each "action" (default 1000)
	default will run the program with 3 doctors and 3 patients with a 1000ms delay
